# Zeenea Connector Downloads

{% callout type="note" title="Important information" %}
- Some plugins require a **minimum scanner version**.
- Download the **latest scanner** from your Zeenea instance:  
  `https://[instance-name].zeenea.app/admin/settings/scanners`
- A **single plugin can contain multiple connectors**.
- Connector configuration templates are available in the  
  [Zeenea connector configuration templates repository](https://github.com/zeenea/connector-conf-templates/tree/main/templates).
- For installation steps, see  
  [Zeenea Scanner Setup](../Scanners/zeenea-scanner-setup.md).
{% /callout %}
